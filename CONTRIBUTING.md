# Contributing

这个仓库只接受“创业和社交场景里的处事规则、原创场景、评测题、方法文档”和 skill 改进。

## 可以提交

- 原创创业现实场景题
- 原创社交现实场景题
- 地推、BD、求人、谈合作、借资源、饭局、客户局、团队局案例
- 同学局、亲戚局、公开夸奖、拒绝、求助、尴尬场面案例
- 对某个场景的行为规则抽取
- 更自然的中文话术模板
- 评测题和通过标准
- 对 skill 触发条件、输出格式、风险边界的改进

## 不要提交

- 小说全文
- 章节正文
- 长段原文摘录
- 可替代阅读原书的剧情整理
- 侵犯版权的资料包

## 新增规则格式

新增规则最好包含：

1. 场景
2. 错误方向
3. 更像陈汉生的动作
4. 为什么这样做
5. 风险边界
6. 可复用规则

重点不是把话说漂亮，而是解释清楚谁有权、谁承担风险、这句话抬了谁、避开了谁、自己获得了什么现实利益。

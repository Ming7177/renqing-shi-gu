# 全书能力地图

## 总体结论

《我真没想重生啊》里适合蒸馏“陈汉生人情世故”的内容，不是剧情复述，而是他在不同场域里反复使用的判断链：

1. 先看场域权力
2. 再定自己的身份
3. 再稳住局面
4. 再给对方台阶
5. 最后把事办成
6. 如果是商业线，再看这一步换到了什么下一阶段资产

这个 skill 只保留可迁移的处事模型，不保留小说正文、长段摘录或完整章节索引。

## 七层场景

### 校园组织与权力

适用场景：
- 班级事务、学生会、辅导员、书记、老师、教官、社团、评优、盖章、竞选。

蒸馏点：
- 先拿身份，再讲事情。
- 对权力方不要只靠嘴催，先判断自己有没有 leverage。
- 自己没身份时，先把自己变成“眼熟、懂事、有用、不添麻烦的人”。
- 要竞选或争取位置时，聚光灯有利，可以接住；但接住时要顺手接责任。

### 同学宿舍与同辈

适用场景：
- 宿舍矛盾、同学互助、同辈比较、聚会抬轿、有人被孤立、有人被难堪。

蒸馏点：
- 同辈关系靠互相帮衬，不靠天然亲密。
- 护自己人时，对内可以批评，对外先保住体面。
- 有人当众被比较时，要先判断这场比较会不会制造敌意。
- 体面比输赢更稳定，能让关系继续转。

### 家庭长辈与亲缘

适用场景：
- 拜访长辈、亲戚饭局、父母场、家庭谈判、第一次上门、长辈评价。

蒸馏点：
- 礼数先完整，事情后再谈。
- 长辈场景里，面子、分寸和称呼是硬约束。
- 不让主家难堪；别人接不住的话，要换一个人、换一种说法接住。
- 家庭局不是纯讲理，是讲时机、身份和见证人。

### 创业商业与合作

适用场景：
- 地推、招商、谈合作、争资源、被质疑、商务谈判、对方试探底线。

蒸馏点：
- 先构建自己的位置，再谈合作。
- 不空口求人，先给对方一个愿意听下去的利益点。
- 不完全按对方脚本走；规则不利时，要重设问题。
- 公开场合要有叙事，不能只有交易。

### 商业发家与资本化

适用场景：
- 校园代理、火箭101、现金流、校园渠道、消费电子、果壳、发布会、竞争、产品上市、公司改制、借壳上市、业务拆分。

蒸馏点：
- 先用低成本场景练手，不一开始就做大盘。
- 每一步都不只赚当前的钱，还要换下一阶段的身份、资源、渠道、人才和信用。
- 火箭101的核心不是快递，而是校园渠道网络和组织能力。
- 果壳的核心不是单个产品，而是品牌叙事、用户身份、舆论阵地、渠道和资本化资产。
- 产品上市和公司上市不是一回事；前者验证市场，后者放大资本。
- 竞争时不要只比参数，要争夺解释权和比较标准。

### 面子人情与台阶

适用场景：
- 饭桌敬酒、拒绝别人、求人办事、替人解围、被夸、被点名、对方不能接话。

蒸馏点：
- 给人台阶就是给自己留路。
- 人情不是虚礼，是长期交换结构。
- 公开表达和私下表达要分层。
- 饭桌上遇到对方不能接酒、不能接话、不能接责任时，不要让话掉地上；找合适的人代接，并把代接包装成赞美。

### 冲突善后与控局

适用场景：
- 顶撞、争执、误会扩大、现场围观、对方要立威、己方人吃亏。

蒸馏点：
- 先止血，再算账。
- 能赔、能拖、能谈、能换叙事，就别一上来硬顶。
- 危机关键不只是事实，而是谁先掌握解释权。
- 让每个人都有退路，局面才可能收住。

### 恋爱修罗场与情绪

适用场景：
- 高情绪关系、误会、吃醋、亲密关系冲突、公开难堪。

蒸馏点：
- 只保留情绪控场、给台阶、保体面、换叙事的方法。
- 不把恋爱线当作做人模板。
- 不把亲密关系当成可以随意调度的资源。

## 公开场合额外规则

当陈汉升在公开场合被聚光灯照到时，优先处理的不是“回答内容”，而是“这束光有没有利”。

如果有利：
1. 接住夸奖
2. 把夸奖转成身份
3. 顺手接责任
4. 借这束光推进目标

如果无利：
1. 先把自己拉回普通位置
2. 编一个足够小、足够现实的临时理由
3. 把话题拉回流程或现实
4. 把对方从围观者拉回帮忙者

核心不是承认或否认，而是不让自己被无收益地比较。

## 反复校准的三条硬规则

1. 聚光灯不是永远要灭。能换来身份、资源、信任、威望时，要接住；会让旁人尴尬、树敌、变成众矢之的时，才灭掉。
2. 饭桌上对方接不住酒或话时，不能只说“算了”。要把尴尬转成面子，找合适的人代接。
3. 面对权力方慢流程，如果你没 leverage，语言催促通常没用。连续低姿态行动，比一句漂亮话更像陈汉生。

# 陈汉生人情世故规则

## 核心判断

陈汉生的人情世故，不是讨好别人，而是先看局势，再定身份，再稳场面，再给台阶，最后把事办成。

## 他真正做的事

- 先看权力，不先看话。
- 先护自己人，不先顾面子作文。
- 该软的时候软，利于转场就软。
- 该硬的时候硬，能改变成本就硬。
- 他交易的是面子、时机、信任和入口，不只是钱。

## 六个可复用动作

1. 定身份
   先决定自己是班长、朋友、晚辈、负责人、自己人还是调解人。

2. 稳场面
   先让气氛降下来，再处理问题。先承认张力，再给一个能继续走的说法。

3. 给台阶
   让对方保住体面。给退出口、软着陆、改口理由。

4. 交换价值
   不要盲求。先给一个好处、一条路、一个资源、一个未来预期。

5. 护自己人
   对外分功，对内扛责，别让自己人单独暴露。

6. 改叙事
   把“谁赢了”改成“什么结果最有用”。

## 语气规则

- 先接住，再推进
- 先半句认同，再往自己要的方向带
- 先打断废话，再落到现实
- 先给面子，再给办法
- 不把话说绝
- 不讲空泛大道理
- 允许自嘲，但不自贬
- 关键判断要直接，别磨叽
- 句子别太长，听起来要像现场在说话

## 节奏规则

1. 先短
2. 再补
3. 再转
4. 再收

意思是：
- 先用一句短话把人接住
- 再补一个事实或理由
- 再把话题带到可执行方向
- 最后留余地，不把门堵死

## 说话动作库

- 接住：先认可一句，别让对方开口就撞墙
- 打断：把空话切掉，不让对话飘走
- 转现实：把抽象话拉回时间、成本、场景
- 给台阶：让对方能退，不把门封死
- 压场：必要时用更硬的一句把局势按住
- 自嘲：少量自嘲，让自己不是高高在上的教训者
- 聚光灯判断：公开场合被夸、被比较、被推到台前时，先算这束光有没有利
- 灭光：聚光灯无利且容易树敌时，编一个小现实，把自己拉回普通人

## 聚光灯规则

遇到别人当众夸你“厉害、独立、特别、少见”时，不要一刀切。

先判断：

1. 这束光能不能给我身份
2. 能不能换来资源或信任
3. 会不会让旁人尴尬
4. 会不会给自己树敌
5. 接住以后能不能把责任也接住

如果有利：
- 接住聚光灯
- 顺势把自己放到能办事的位置
- 同时接一点责任，别只接夸奖

如果无利：
- 先否掉特殊性
- 给一个临时小理由
- 把自己拉回普通人
- 把对方拉回帮忙者

灭光例子逻辑：
- 不是我厉害，是家里临时有事
- 不是我特别，是本来父母也要来
- 不是我独立，是我先过来跑一趟
- 后面流程还得麻烦你

## 代劳转赞美规则

饭桌、长辈局、家庭局里，如果你敬的人不能接，不要只说“那算了”。

正确动作：

1. 先承认被敬者的功劳
2. 不让被敬者承担压力
3. 找一个合适的家人代接
4. 把“代喝”包装成“替家人分忧”
5. 顺手夸代接者已经长大、懂事、能担事

这样一来，你不是少敬一杯，而是同时完成三件事：

- 敬到了女主人
- 给了儿子面子
- 让男主人觉得家里被抬了

模板：

`嫂子今天忙这一桌，肯定不能再让她喝。小海也成年了，肩膀能替母亲分点事了，这杯就请你代劳。`

## 买单买局规则

饭局里遇到账单、买单、被人看向你时，不要第一反应就是“我该不该付钱”。先判断这顿钱能不能买到一个局。

先看四件事：

1. 这桌人对我有没有现实价值
2. 我买单能不能换到身份、人脉、资源、联系方式或下一次见面
3. 买单后我是不是能把场面变成自己的局
4. 如果我不买，会不会损失重要关系或关键面子

如果能换到东西：

- 可以主动买
- 但买完要把局接过来
- 要让别人知道这是你在收场，不是你被推着当冤大头
- 顺手留下联系方式、下次饭局入口或合作口子

模板逻辑：

`行了，账我先结。今天别因为一顿饭扫兴，人不见了我回头找他。大家先把场面收住，以后有机会我再单独请各位坐坐。`

如果换不到东西：

- 不接全责
- 只接协调权
- 把账单摊开，按共同参与规则处理
- 自己最多认和自己身份匹配的一份

模板逻辑：

`账单我先看一下。我和他熟，所以我来协调，不代表今晚我做东。今天大家都是来给他过生日的，不是来给我过生日的。先按人头结，寿星那份我垫。`

这条规则的核心是：

> 买单不是花钱，是买局。没局可买，就别买。

## 亲属争执撤退规则

当你在别人家做客、求人、拜访长辈或维护关系时，如果对方和亲人突然吵起来，不要急着展示自己会圆场。

先判断三件事：

1. 我是不是这个家庭内部的人
2. 我有没有资格评价谁对谁错
3. 我留下来会不会变成以后被翻出来的证人和裁判

如果你是外人、晚辈、客人，正确动作通常是撤退。

撤退不是不讲情面，而是不给自己加麻烦。亲人今天吵，明天可能和好；外人今天听，明天就可能被问“你当时也在，你说谁有理”。

正确动作：

1. 不劝架
2. 不站队
3. 不点评
4. 给一个不伤人的退出理由
5. 把空间还给他们家里人
6. 改天再单独处理自己的事

模板逻辑：

`老师，那我先不打扰您和家里人说事了。今天我本来就是来麻烦您的，别因为我在这儿让您不方便。我改天再过来。`

这条规则的核心是：

> 疏不间亲。能撤就撤，别让自己变成以后他们复盘矛盾时的证人和裁判。

## 付出记忆规则

关系里最容易被误判的一件事是：以为自己对别人越好，别人越放不下你。

现实往往不是这样。单方面跋山涉水、随叫随到、一直顺路，容易让对方把你当成稳定供给。对方记住的未必是你付出了多少，而是自己有没有为这段关系付出过成本。

先判断四件事：

1. 这段关系里谁一直在付出
2. 对方有没有主动选择过我
3. 对方有没有为我花过时间、面子、资源或情绪成本
4. 我的好是不是已经变成对方不需要珍惜的默认待遇

正确动作不是停止对人好，而是不要把关系做成单向供养。

可用动作：

1. 不做随叫随到的无底线供给
2. 适当让对方帮一个小忙
3. 让对方参与选择，而不是只接受安排
4. 让对方为关系投入时间、行动、承诺或资源
5. 对方投入后要接住，让他觉得“这是我选的、我做的、我也在里面”

这条规则在创业和社交里都能用：

- 求人：不要只送礼说好话，要让对方有一个参与和成全你的理由。
- BD：不要只免费服务，要让客户投入一点数据、时间、试点资源。
- 朋友：不要把自己变成永远顺路的人，要让对方也有主动走向你的时刻。
- 团队：不要只靠老板发福利，要让成员参与建设，项目才会变成“我们的事”。

模板逻辑：

`别光自己往前凑。你得让对方也伸一次手，哪怕只是一个小忙。人很多时候记住的不是谁对他好，而是他自己为谁花过心思。`

这条规则的核心是：

> 别只做供给者，要让对方也投入。投入越多，越容易把这段关系写进自己的账本。

## 场面价值帮忙规则

帮忙不是道德考试，而是场面判断。不是见事就上，也不是见事就躲。

先判断五件事：

1. 被帮的人是不是主家、自己人、关键人物或未来入口
2. 这件事如果没人接，会不会让场面难看
3. 我出手的成本是不是可控
4. 我出手后能不能稳住主家、抬住对方、让旁人看见我会来事
5. 这个忙会不会变成接锅、背债、当裁判或惹麻烦

如果是低成本、高场面价值，就别装不认识。

正确动作：

1. 先看主家会不会难堪
2. 再看自己能不能低成本补位
3. 出手时别邀功，像顺手一样接过去
4. 帮完就退回场面，不把功劳挂嘴上

如果帮了会接锅，就不要乱心软。

模板逻辑：

`这忙可以帮。不是因为我和他多熟，是主家这会儿不能塌场。我过去挡一下，成本不大，人家心里会有数。`

这条规则的核心是：

> 姿态要跟场合匹配：该守边界就守边界，该融入就融入；该补位就补位，该抽身就抽身。

## 权力方慢流程规则

当你有求于学校、单位、组织里的权力方，对方用“流程”拖着你时，不要只靠话术催。

如果你没有把柄、没有交换资源、没有身份优势，硬问截止时间只会让对方拖到最后一刻。

正确动作：

1. 不在嘴上催
2. 连续出现
3. 做低姿态小服务
4. 让对方知道你有所求，但不把话挑明
5. 等对方主动问
6. 问到时只讲真实困难，不讲威胁

这招的核心不是讨好，而是把自己从“外人”变成“眼熟、懂事、不会添麻烦的人”。

对方不是不知道你的小算盘。恰恰因为知道，又看到你连续几天做事，才有理由把章盖了。

模板逻辑：

- 不说“书记你什么时候盖”
- 改成连续几天帮他整理办公室、擦桌子、分资料
- 等书记主动问“你们班那个人真贫困吗”
- 再回答事实：“真的，书记，他每天吃馒头喝面汤。”

## 场景映射

- 饭局：先看谁是主位，别让主家难堪，先稳桌面。
- 买单：先算能不能买到局；能买局就买，买完接场；买不到局就只协调，不接冤枉账。
- 求人：先让请求显得值得听，再开口。
- 拒绝：拒绝，但别让对方丢脸。
- 亲戚局：礼数要完整，但别让情分盖过判断；亲属争执里外人不要抢裁判位。
- 同学局：把地位表演当背景噪音，用体面和时机控场。
- 商务谈判：别照对方脚本走，规则不好就重设牌桌。
- 关系经营：别只单向付出，要看对方有没有投入过时间、选择、行动或资源。
- 帮忙补位：先看场面价值；低成本稳主家就上，会接锅惹麻烦就退。
- 发家路径：别只看当前赚钱，看这一步换到了什么下一阶段资产。
- 冲突善后：先止血，再决定赔、拖、谈、换叙事。

## 发家路径规则

陈汉生从校园代理走到果壳，不是一路“做大同一件事”，而是连续换牌桌。

判断一个创业动作有没有价值，不只看它现在赚多少钱，还要看它换到了什么：

1. 身份
2. 资源
3. 现金流
4. 渠道
5. 团队
6. 用户心智
7. 解释权
8. 下一张牌桌的门票

关键规则：

- 学生身份不是弱点，可以变成学校背书和组织入口。
- 小生意不是终点，要看能不能训练组织、沉淀渠道、积累信用。
- 抢代理不是靠喊口号，要拿一个对方能接受的替换标准。
- 火箭101的本质不是快递，是校园渠道网络。
- 果壳的本质不是单个电子产品，是品牌、舆论、粉丝和渠道的组合。
- 产品上市解决市场验证，公司上市解决资本放大。
- 做到后期，要从“卖产品”切到“管理资产组合”。

## 不要做的事

- 不要写成纯善良。
- 不要写成纯坏人。
- 不要把他写成口号机。
- 不要把每个动作都写成万能套路。
- 不要把亲密关系当成可以随意调度的资源。

## 回答顺序

被问到怎么模仿或分析时，按这个顺序答：

1. 场景
2. 压力点
3. 陈汉生动作
4. 原因
5. 风险
6. 可复用规则

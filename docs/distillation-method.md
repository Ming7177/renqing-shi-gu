# 蒸馏方法

## 目标

不是复刻剧情，而是抽出可执行的处事模型。

## 分析顺序

1. 场景归类
   - 校园组织与权力
   - 同学宿舍与同辈
   - 家庭长辈与亲缘
   - 创业商业与合作
   - 面子人情与台阶
   - 冲突善后与控局
   - 恋爱修罗场与情绪

2. 动作抽取
   - 定身份
   - 稳场面
   - 给台阶
   - 交换价值
   - 护自己人
   - 改叙事

3. 说话方式抽取
   - 接住
   - 打断
   - 转现实
   - 给台阶
   - 压场
   - 自嘲

4. 场景测试
   用户给现实场景，skill 盲答，再用原书类似场景校准。

## 关键原则

- 不把人物道德标签放在第一位。
- 不把所有场景都写成油滑话术。
- 不复制长段原文。
- 每次新增规则，都要来自具体场景失败后的修正。

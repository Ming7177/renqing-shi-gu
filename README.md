一个把《我真没想重生啊》中陈汉升的人情世故处事方式，蒸馏成可复用 Codex skill 的开源项目。

它不复述剧情，不搬运原文，也不教人背漂亮话。它只做一件事：

> 把一个复杂现实场景，拆成“谁有权、谁要面子、谁需要台阶、我能换什么、最后怎么把事办成”。

## 先拿场景来考它

别先看文档，先拿这三道题试：

1. 学姐当众夸你一个人报到很独立，旁边父母陪同的同学都尴尬了，你怎么接？
2. 女主人做了一大桌菜，你敬她酒，她说不会喝，桌上还有男主人和成年儿子，你怎么转？
3. 书记一直拖着不给贫困生材料盖章，每次都说流程还在走，你怎么处理？

普通 AI 会给你“礼貌、真诚、表达感谢”。这个 skill 要回答的是：

- 这束聚光灯该接还是该灭
- 这句话抬了谁，避开了谁
- 你现在是求人、晚辈、同学、负责人，还是调解人
- 哪句话只是开口，哪件事才是真动作

## 为什么做这个

很多 AI 建议听起来都很正确，但一到饭桌、学校、求人、地推、同学局、亲戚局，就立刻露出 AI 味：

- 只会说“真诚沟通”
- 只会说“表达感谢”
- 只会说“保持边界”
- 不知道谁会尴尬
- 不知道谁需要面子
- 不知道什么时候该接聚光灯，什么时候该把光灭掉

这个 skill 的目标不是让回答更礼貌，而是让回答更像一个真正在现场办事的人。

## 它和普通 AI 的区别

| 场景 | 普通 AI | 人情世故 |
| --- | --- | --- |
| 当众被夸 | 谢谢夸奖，我会继续努力 | 先算这束光有没有利；无利就把自己拉回普通人 |
| 饭桌敬酒遇到对方不能喝 | 那就以茶代酒吧 | 先保女主人，再把酒转给合适家人，顺手夸人 |
| 求权力方盖章被拖 | 说明截止时间，请对方尽快处理 | 没 leverage 时不嘴上催，先连续出现，变成眼熟又懂事的人 |
| 地推被商家拒绝 | 介绍产品优势 | 先给对方一个愿意听 30 秒的现实利益 |

## 一眼看懂

场景：

> 新生报到时，学姐夸你一个人来很独立。旁边很多父母陪同的学生表情尴尬。

普通 AI 容易回答：

> 谢谢学姐夸奖，我从小比较独立。

这会让你直接变成众矢之的。

“人情世故”会先判断聚光灯有没有利。这个场景没有利，还会让旁边人难堪，所以要把自己拉回普通人：

> 没有学姐，本来父母也要来的，车票都买好了。家里临时有事走不开，我就先自己过来了。后面流程还得麻烦你带一下。

这不是“会说话”，这是把风险、面子、流程一次处理掉。

## 核心模型

陈汉升不是简单的“会说话”。他的核心是现实判断：

1. 先看场域权力
2. 再定自己的身份
3. 再稳住局面
4. 再给对方台阶
5. 最后把事办成

更短一点：

```text
看局势 -> 定身份 -> 稳场面 -> 递台阶 -> 办成事
```

## 能处理什么

- 饭局：谁是主位，谁不能难堪，酒怎么转，话怎么接。
- 求人：自己没 leverage 时，怎么先变成“眼熟、有用、不添麻烦的人”。
- 拒绝：话可以拒绝，但不能把对方路堵死。
- 同学局：有人抬你、贬别人时，怎么不接无收益的聚光灯。
- 亲戚局：礼数、称呼、见证人、时机，哪个先动。
- 商务谈判：不按对方脚本走，必要时重设问题。
- 创业地推：先给对方愿意听下去的理由，而不是上来推销。

## 安装

把 `skill/` 目录复制到你的 Codex skills 目录下，并改名为：

```text
人情世故
```

Windows 示例：

```powershell
Copy-Item -Recurse .\skill D:\.codex\skills\人情世故
```

然后重启 Codex，让技能重新扫描。

## 使用

触发名：

```text
$人情世故
```

示例：

```text
用 $人情世故 分析这个场景：我创业初期要去地推，商家不信任我，我该怎么开口？
```

它会优先给你三件事：

1. 先看什么
2. 先说什么
3. 先做什么

可以直接复制这段测试：

```text
用 $人情世故 回答：

我创业初期去做地推，进一家小店。老板正在忙，不太想理我，说“我们这里不需要这些东西”。我不想被一句话打发，也不想显得像推销员。这个时候先看什么、先说什么、先做什么？
```

## 贡献方式

欢迎提交新的原创场景题、规则抽取、评测题和更自然的话术模板。

新增规则最好回答清楚：

1. 这句话抬了谁
2. 避免伤了谁
3. 自己拿到了什么现实利益
4. 哪一步是风险边界

## 边界

这个仓库不包含小说原文、不包含章节正文、不包含长段摘录，也不试图替代阅读原书。

这里开源的是读完之后抽出来的行为模型、原创示例和评测方法。

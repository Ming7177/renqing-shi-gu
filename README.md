很多 AI 建议听起来都正确，但一进现实场景就没用。

它会告诉你“真诚沟通、表达感谢、保持边界”，但不会告诉你：

- 这局谁更有权
- 谁要面子
- 谁需要台阶
- 哪束聚光灯该接，哪束光该灭
- 你现在能交换什么
- 先说什么，先做什么

`人情世故` 想解决的就是这个问题。

它把《我真没想重生啊》中陈汉升的现实判断，蒸馏成一个 Codex skill。陈汉升是来源，真正要复用的是一套现实场景方法：

```text
看局势 -> 定身份 -> 稳场面 -> 递台阶 -> 换价值 -> 办成事
```

## 先拿两类题试

### 创业题

```text
用 $人情世故 回答：

我创业初期去做地推，进一家小店。老板正在忙，不想听我介绍，说“我们这里不需要这些东西”。我不想被一句话打发，也不想显得像推销员。这个时候先看什么、先说什么、先做什么？
```

它不应该上来介绍产品优势，而是先判断：

- 老板怕被打扰、被骗、浪费时间
- 我现在不是供应商，而是先帮他看问题的人
- 第一目标不是成交，是让老板愿意给 30 秒

可用接法：

> 老板，我不耽误你做生意。你先忙，我就说一句：我不是来让你马上买东西的，我先看你店里哪一步最费人。要是帮不上忙，我自己走，不占你时间。

### 社交题

```text
用 $人情世故 回答：

我去学校报到，学姐当众夸我一个人来很独立。旁边很多父母陪同的同学都尴尬了。这个时候我怎么接？
```

它不应该顺着夸自己独立，而是先判断这束聚光灯有没有利。

这个场景没有利，还会让旁边人难堪，所以要把自己拉回普通人：

> 没有学姐，本来父母也要来的，车票都买好了。家里临时有事走不开，我就先自己过来了。后面流程还得麻烦你带一下。

## 三条主线

### 创业线

| 场景 | 普通 AI | 人情世故 |
| --- | --- | --- |
| 地推被拒 | 介绍产品优势 | 先降低老板防备，让他愿意听 30 秒 |
| 没案例谈试点 | 强调未来价值 | 把合作拆成低风险、短周期、可验证的小试点 |
| 求人介绍资源 | 表达诚意和感谢 | 先让对方有转介绍的理由和台阶 |
| 被客户压价 | 强调成本和价值 | 先稳住对方身份，再重设比较标准 |
| 现金流紧张要延期 | 解释困难 | 给对方安全感、时间点和补偿 |

### 发家线

| 阶段 | 表面动作 | 真正换到的东西 |
| --- | --- | --- |
| 班长和学生会 | 做校园组织事务 | 合法身份、老师信任、组织入口 |
| 深通校园代理 | 抢快递总代理 | 低成本现金流、校园场景、替换标准 |
| 火箭101 | 做校园快递网络 | 渠道、学生队伍、跨校复制能力 |
| 果壳电子 | 做MP4和手机 | 品牌、产品、供应链、年轻用户心智 |
| 手机发布会 | 推产品上市 | 舆论场、粉丝、解释权、市场验证 |
| 果壳网络和收购 | 做生态产品 | 独立估值、资本化路径、上市想象 |

### 社交线

| 场景 | 普通 AI | 人情世故 |
| --- | --- | --- |
| 当众被夸 | 谢谢夸奖，我会继续努力 | 先算这束光有没有利；无利就灭光 |
| 饭桌敬酒遇到对方不能喝 | 那就以茶代酒吧 | 先保女主人，再转给合适家人，顺手夸人 |
| 饭局账单看向你 | 该不该AA | 先算这顿钱能不能买到局；能买就接场，不能买只协调 |
| 同学局被人抬高 | 谦虚一下 | 看是否会伤旁人，必要时把自己拉回普通人 |
| 求权力方办事被拖 | 说明截止时间，请尽快处理 | 没 leverage 时不嘴上催，先变成眼熟又懂事的人 |
| 亲属争执要你旁听 | 留下劝架显得懂事 | 疏不间亲，外人先撤，别变成未来裁判和证人 |
| 单方面对人好不被珍惜 | 继续加码真诚 | 别只做供给者，要让对方也投入时间、选择或资源 |
| 喜事场有人撑不住 | 和我不熟，不管 | 先看场面价值，低成本稳主家就上 |
| 亲戚局被问收入 | 含糊带过 | 给长辈面子，也不让同辈被比较 |

## 适合谁

- 创业早期创始人
- 做地推、BD、招商、渠道合作的人
- 经常求人办事、找资源、谈介绍的人
- 经常进入饭局、亲戚局、同学局、客户局的人
- 想训练 AI agent 现实场景判断力的人

## 不是什么

- 不是小说原文库
- 不是话术大全
- 不是PUA工具
- 不是教你装聪明
- 不是让所有人都喜欢你

它只解决一个问题：在复杂的人情场里，先别把局走死。

## 安装

把 `skill/` 目录复制到你的 Codex skills 目录下，并改名为：

```text
人情世故
```

Windows 示例：

```powershell
Copy-Item -Recurse .\skill D:\.codex\skills\人情世故
```

然后重启 Codex，让技能重新扫描。

## 使用

触发名：

```text
$人情世故
```

示例：

```text
用 $人情世故 分析：

我想找一个本地商家合作做试点，但我现在没案例、没品牌、没预算。老板觉得我不靠谱。我该怎么开口，怎么让他愿意给我一次小测试？
```

它应该优先输出三件事：

1. 先看什么
2. 先说什么
3. 先做什么

## 贡献方式

欢迎提交创业场景、社交场景、BD话术拆解、饭局/求人/合作案例、评测题和更自然的中文表达。

新增规则最好回答清楚：

1. 这局谁更有权
2. 谁承担风险
3. 这句话抬了谁
4. 避免伤了谁
5. 自己拿到了什么现实利益
6. 哪一步是风险边界

## 边界

这个仓库不包含小说原文、不包含章节正文、不包含长段摘录，也不试图替代阅读原书。

这里开源的是读完之后抽出来的行为模型、原创场景和评测方法。

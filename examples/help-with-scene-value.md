# 示例：婚礼上主家亲属被灌酒，要不要帮

## 场景

你参加一场婚礼，和主家不算熟。席间主家一个年轻亲属被人轮番敬酒，已经明显撑不住了。

旁边朋友劝你：“你又不认识他，帮这个忙做什么？”

## 错误方向

只按“认不认识”判断。

如果你只看熟不熟，就会错过场面价值。婚礼是主家的喜事，主家亲属当众被灌倒，难看的不是他一个人，是主家的场面。

## 陈汉升式处理

先看这个忙值不值得帮。

这局里：

- 被帮的人是主家一边的人
- 场面正在变难看
- 你挡两杯的成本可控
- 主家会记得你替他们稳了一下
- 旁边人也能看见你会补位

所以可以上。

可用接法：

“我去挡两杯。不是和他多熟，是主家这会儿不能塌场。两杯酒的事，换个人情，不亏。”

如果朋友还拦：

“你别光看认不认识。喜事场上，主家难看，客人也没面子。我过去接一下，喝不了就撤。”

## 为什么这样做

这不是滥好人，而是低成本稳场。

有些忙不该帮，因为会接锅、背债、当裁判。有些忙该帮，因为帮了能稳主家、抬对方、让自己获得一个好印象，而且成本很低。

## 规则

不是帮不帮，是看这个忙有没有场面价值。

低成本、高场面价值、能稳主家的忙，可以帮。

会接锅、背债、惹麻烦、让自己变成责任人的忙，不要乱心软。
